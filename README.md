# HACKING ON ZIG


## Build all Targets

```bash
$ zig build # build all targets
```

## Run a Target

```bash
$ zig build run_hello # run a target
```

or

```bash
$ zig build run_aoc1 -- ./src/aoc1/input.txt # run a target with input
```

alternatively

```bash
$ ./zig-out/bin/aoc1 ./src/aoc1/input.txt # run a target with input
```
